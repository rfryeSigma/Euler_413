The code in the modular sub-folder attempts to find relationships
between elliptical curves with modular curves and forms and other transformsations.

jacobian_mumford.py-- AI suggestion on Mumford representation of Jacobian of D2 quartic

lll_approx.py -- AI suggestion on LLL approximation to A^4+B^4+C^4=D^4+eps

lll_quad.py -- AI suggestion on LLL approximation to (x^2)^2 + (y^2)^2 + (z^2)^2 = 1

modular.log -- AI suggestion on modular curves, used to code solutions_modular.py

solutions_modular.py -- find solutions by various searches.

picard.py -- AI suggestion to treat K3 surface with a known Picard lattice

uv_recurrence.py -- AI suggestion for LLL pattern match on known (u, v). FAILED.
